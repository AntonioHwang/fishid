# Fish Identification Model

사진 속 물고기의 이름(또는 널리 알려진 별명)이 무엇인지 알려주는 모델.

# Dataset

- 대상 물고기 목록 파일은 data/fish_list.txt 참고

- 더 많은 물고기 목록 파일은 data/fish_list_all.txt 참고

- 구글 크롤링 -> image classification model로 물고기 이미지만 필터링 -> 최종 인간지능 정제

- 현재 물고기 목록 정 중
